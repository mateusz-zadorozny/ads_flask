How to get the right file?

Currently the tool accepts only Facebook Ads Manager .csv files created while your account is set to english.

  1. Select one campaign or ad set.
  2. Then use the filter: Selection only.
  3. After you see only one campaign or ad set, use the export feature.
  4. Choose the .csv file format.
